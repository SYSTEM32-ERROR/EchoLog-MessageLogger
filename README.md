## ✨ Features

* 🗑️ **Deleted Messages**

  * Keeps deleted messages visible in chat
  * Highlighted with clean red styling

* ✏️ **Edit History**

  * Tracks message edits in real time
  * Shows previous versions inline

* 👻 **Ghost Ping Detection**

  * Detects deleted or edited mentions
  * Instant alerts when someone ghost pings you

* 🔔 **Smart Notifications**

  * Toasts for deletes, edits, purges, and ghost pings

* 🎨 **Modern Styling**

  * Color-coded events:

    * Deleted → Red
    * Edited → Orange
    * Purged → Gray
    * Ghost Ping → Blue

* 🧠 **Advanced Filtering**

  * Ignore bots, muted servers, blocked users, etc.
  * Whitelist / blacklist support

* 🖼️ **Image Caching**

  * Saves deleted images locally

* 📂 **Log Viewer UI**

  * Clean built-in viewer with filters

* ⌨️ **Quick Access**

  * `Ctrl + M` shortcut
  * Toolbar button
  * Context menu integration

## ⚙️ Installation

1. Install **BetterDiscord**
2. Download `EchoLog.plugin.js`
3. Move it to:

   ```
   %appdata%/BetterDiscord/plugins
   ```
4. Enable it in Discord settings

---

## 📦 Dependencies

EchoLog requires:

* ZeresPluginLibrary
* XenoLib

The plugin will wait until they are loaded.

---

## ⚙️ Configuration

EchoLog includes a full settings panel:

* Ignore bots / self / blocked users
* Ignore muted servers or channels
* Toggle logging types
* Edit history limit
* Message storage cap
* Whitelist / blacklist control
* Custom event colors

---

## 🧠 How It Works

EchoLog hooks into Discord’s internal message system to:

* Capture messages before deletion
* Store edit history
* Re-inject deleted messages into chat
* Apply live styling
* Detect ghost pings using mention tracking

---

## 🖱️ Usage

* Press **Ctrl + M** → Open logs
* Click toolbar icon → Open viewer
* Click highlighted messages → Jump to logs
* Right-click → Access quick options

---

## 📁 Data Storage

Stored locally using BetterDiscord:

* Message logs
* Edit history
* Ghost ping data
* Cached images
